### Unix

如果你使用 `Linux` 或 `macOS`， 你可以输入一下命令安装:

```shell
# 安装最新版本
curl -fsSL https://raw.githubusercontent.com/axetroy/gmock/master/install.sh | bash
# 安装指定的版本
curl -fsSL https://raw.githubusercontent.com/axetroy/gmock/master/install.sh | bash -s v0.1.0
```
